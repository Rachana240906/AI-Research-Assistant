# AI Research Assistant (LangChain Agent)

An intelligent research assistant built using LangChain and OpenAI that can:
• Answer questions
• Perform multi-step reasoning
• Use Wikipedia as an external knowledge source
• Generate structured outputs

---

## Tech Stack
- Python
- LangChain
- OpenAI GPT models
- Wikipedia Tool
- Pydantic

---

## Features
- Tool-calling agent
- External knowledge retrieval
- Multi-step reasoning
- CLI-based interaction
- Modular design

---

## Setup

### 1. Clone repo
git clone https://github.com/Rachana240906/AI-Research-Assistant.git
cd AI-Research-Assistant

### 2. Install dependencies
pip install -r requirements.txt

### 3. Add API key
Create `.env` file:
OPENAI_API_KEY=your_key_here

### 4. Run
python main.py

---

## Example
Input:
What is reinforcement learning?

Output:
Explanation generated using LLM + Wikipedia search
